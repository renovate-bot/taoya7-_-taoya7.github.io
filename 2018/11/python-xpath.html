Xpath XML Path Language

XPath常用规则

nodename          选取此节点的所有子节点
/                 从当前节点选取直接子节点
//                从当前节点选取子孙节点
.                 选取当前节点
..                选取当前节点的父节点
@                 选取属性

所有节点

注意

  • 星号 表示所有节点
  • 返回形式是一个列表，每个元素是Element类型 后面跟了节点的名称
from lxml import etree

html = etree.parse('Test.html', etree.HTMLParser())
result = html.xpath('//*')

子节点

注意
/ 或 // 可以查找子节点或子孙节点

<ul class="You can see me">
    <li>1</li>
        <a href="#">Tashi</a>
    <li>2</li>
    <li>3</li>
    <li>4</li>
    <li>5</li>
</ul>

# 获取ul节点下的li节点
result = html.xpath('//ul/li')
print(result)

父节点

<ul class="You can see me">
    <li>1</li>
        <a href="#">Tashi</a>
    <li>2</li>
    <li>3</li>
    <li>4</li>
    <li>5</li>
</ul>


result = html.xpath('//a[@href="#"]/../@class')
print(result)  # 选中href=# 的a节点， 然后获取父节点 再获取其class的属性

> You can see me

result = html.xpath('//a[@href="#"]/parent::*/@class')
print(result)
>>> You can see me

属性匹配

<ul>
    <li>1</li>
    <li class="item-3">2</li>
</ul>

result = html.xpath('//li[@class="item-3"]')

文本获取

<ul>
    <li>1</li>
    <li class="item-3">2</li>
</ul>

result = html.xpath('//li[@class="item-3"]/text()')

>>> 2
